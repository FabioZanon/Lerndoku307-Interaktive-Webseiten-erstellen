Was ist JavaScript?

JavaScript ist keine Programmiersprache, sondern eine Scriptsprache. Sie wird dafür verwendet, um HTML Elemente zu manipulieren. Das bedeutet, dass man den Elementen angibt, wie sie sich verhalten sollen unter einer oder mehreren Bedingungen. Symbol:

JavaScript einbinden

Wie auch CSS, muss JavaScript erst einmal eingebunden werden. Anders wie bei CSS, sollte man die Einbindung kurz vor dem schliessendem "body-Tag" platzieren. Wie bereits oben erwähnt, werden HTML Elemente manipuliert. Da der Computer den Code von oben nach unten liest, muss ja ein HTML Element zuerst erscheinen, damit es manipuliert werden kann. Ansonsten hätte JavaScript es mit einem Element zu tun, welches noch gar nicht erschienen ist.

Syntax für das Einbinden: <script src="script.js"></script>

Funktion & Verwendung von i++

Die Funktion "i++" wird in den so genannten "Schleifen" verwendet. Dabei steht "i++" für nichts anderes, als "i = i+1" In einer Schleife wird eine Bedingung geprüft. Wenn sie "true" ergibt, wird ein Codeblock ausgreführt. Anschliessend wird die Bedingung erneut geprüft, und wenn sie immer noch true ist, wird der Codeblock erneut ausgeführt. Das wird so lange wiederholt, bis die Prüfung false zurückgibt. Es gibt drei Arten von Schleifen:

  • FOR
  • WHILE
  • DO..WHILE

For-Schleife: Wenn der Code in einer bestimmten Anzahl von Durchläufen ausgeführt werden muss, verwendet man eine for-Schleife. (Gebräuchlichste Art von Schleife.) Die Bedingung in einer for-Schleife ist ein Zähler, der angibt, wie oft die Schleife durchlaufen werden soll. Hier ist die Syntax für eine for-Schleife:

Gezeigt wird hier eine Variable mit dem i-Wert von 0. Die Aufgabe dieser Schleife ist nun, sie solange zu wiederholen, bis die Zahl 4 erreicht wird. Denn i soll ja kleiner sein als 5.


While-Schleife: Wenn man nicht weiss, wie oft der Code ausgeführt werden muss, verwendet man eine while-Schleife. Die Bedingung muss hier kein Zähler sein. Der Code durchläuft die Schleife so lange, wie die Bedingung true ist.

Do..While-Schleife. Die do..while-Schleife kann man gut mit der while-schleife vergleichen. Nur dient die Do..While-Schleife dafür, dass sie IMMER mindestens einmal ausgeführt wird, selbst wenn die Bedingung gleich zu Anfang zu false ausgewertet wird.
