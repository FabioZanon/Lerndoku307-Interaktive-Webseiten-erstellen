Was ist JavaScript?

JavaScript ist keine Programmiersprache, sondern eine Scriptsprache. Sie wird dafür verwendet, um HTML Elemente zu manipulieren. Das bedeutet, dass man den Elementen angibt, wie sie sich verhalten sollen unter einer oder mehreren Bedingungen. Symbol:

JavaScript einbinden

Wie auch CSS, muss JavaScript erst einmal eingebunden werden. Anders wie bei CSS, sollte man die Einbindung kurz vor dem schliessendem "body-Tag" platzieren. Wie bereits oben erwähnt, werden HTML Elemente manipuliert. Da der Computer den Code von oben nach unten liest, muss ja ein HTML Element zuerst erscheinen, damit es manipuliert werden kann. Ansonsten hätte JavaScript es mit einem Element zu tun, welches noch gar nicht erschienen ist.

Syntax für das Einbinden: <script src="script.js"></script>
